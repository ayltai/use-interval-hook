## Installation
```shell script
npm install @ayltai/use-interval
```

## Usage
```js
useInterval(callback, interval);
```

## Parameters
| Parameter | Description                                                     |
|-----------|-----------------------------------------------------------------|
| `callback` | The function that will be called every `interval` milliseconds |
| `interval` | A number that represents the interval                          |
